The repository is to run the FCNV3 in nano5 in Taiwan HPC.

The reference is from earth2studio, FCNV3, and makani.

https://github.com/NVIDIA/earth2studio

https://github.com/NVIDIA/makani

https://developer.nvidia.com/blog/fourcastnet-3-enables-fast-and-accurate-large-ensemble-weather-forecasting-with-scalable-geometric-ml/

The repository provides the step from building the environment by conda or uv and running the FCNV3 code on nano5.
